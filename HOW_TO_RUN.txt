# 🎮 LINE Chat — نسخه تک‌فایلی

همه‌چیز (سرور، دیتابیس، صفحه‌ی چت) توی یک فایل به اسم **`run.py`** هست. فقط یه پکیج باید نصب کنی.

## ✅ مراحل (روی ویندوز، مک، یا لینوکس یکسانه)

### ۱. پایتون نصب کن (اگه نداری)
از اینجا دانلود کن: https://www.python.org/downloads
(روی ویندوز موقع نصب حتما تیک **"Add Python to PATH"** رو بزن)

### ۲. ترمینال/CMD رو باز کن و بیا توی پوشه‌ای که `run.py` توشه

### ۳. پکیج لازم رو نصب کن
```
pip install aiohttp
```

اگه این خطا رو گرفتی:
```
error: externally-managed-environment
```
به‌جاش این رو بزن:
```
pip install aiohttp --break-system-packages
```

### ۴. سرور رو روشن کن
```
python run.py
```

اگه `python` کار نکرد، `python3` رو امتحان کن:
```
python3 run.py
```

### ۵. برو تو مرورگر
آدرس زیر رو باز کن:
```
http://localhost:8080
```

تمام! حالا ثبت‌نام کن و چت کن. **اولین نفری که ثبت‌نام کنه خودکار ادمین می‌شه.**

---

## ❓ چیزی که نصب نشد / ارور خوردی؟
بهم بگو دقیقاً چه پیامی رو دیدی، تا دقیق بگم باید چیکار کنی.

## 🌐 می‌خوای برای همه روی اینترنت آنلاینش کنی؟
این نسخه برای اجرای **محلی روی کامپیوتر خودت** ساده‌ترینه. برای دیپلوی رایگان روی اینترنت (Render)، نسخه‌ی چندفایلی پروژه بهتره — بگو تا اون نسخه رو هم برات بفرستم.
