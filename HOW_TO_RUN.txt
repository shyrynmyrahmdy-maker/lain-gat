# 🎮 LINE Chat — نسخه تک‌فایلی (به‌روزرسانی شده)

همه‌چیز توی یک فایل به اسم **`run.py`** هست. فقط یه پکیج باید نصب کنی.

## ✅ مراحل اجرا

### ۱. پایتون نصب کن (اگه نداری)
https://www.python.org/downloads
(روی ویندوز موقع نصب حتما تیک **"Add Python to PATH"** رو بزن)

### ۲. ترمینال/CMD رو باز کن، بیا توی پوشه‌ای که `run.py` توشه

### ۳. پکیج لازم رو نصب کن
```
pip install aiohttp
```
اگه این خطا رو گرفتی: `error: externally-managed-environment`
به‌جاش بزن:
```
pip install aiohttp --break-system-packages
```

### ۴. سرور رو روشن کن
```
python run.py
```
(اگه کار نکرد: `python3 run.py`)

### ۵. برو تو مرورگر
```
http://localhost:8080
```

## 🔑 کد ادمین (مهم!)

برخلاف قبل، **دیگه اولین نفری که ثبت‌نام کنه خودکار ادمین نمی‌شه.** فقط کسی که کد مخصوص رو بدونه می‌تونه ادمین بشه:

- **کد پیش‌فرض:** `line-admin-2026`
- موقع ثبت‌نام یه فیلد «کد ادمین (اختیاری)» هست — این کد رو اونجا بزن تا ادمین بشی.
- اگه قبلاً بدون کد ثبت‌نام کردی، می‌تونی از پروفایلت (روی اسم خودت کلیک کن) کد رو بعداً وارد کنی.

**🔒 پیشنهاد می‌کنم این کد رو عوض کنی** تا کس دیگه‌ای نتونه ادمین بشه. دو راه داری:

**راه ۱ (ساده‌تر):** فایل `run.py` رو با Notepad یا هر ادیتور متنی باز کن، دنبال این خط بگرد:
```python
ADMIN_CODE = os.environ.get("ADMIN_CODE", "line-admin-2026")
```
و `"line-admin-2026"` رو با کد خودت عوض کن.

**راه ۲ (بدون دستکاری فایل):** قبل از اجرا کردن، این environment variable رو ست کن:
```
# ویندوز (CMD):
set ADMIN_CODE=my-secret-code
python run.py

# مک/لینوکس:
ADMIN_CODE=my-secret-code python run.py
```

## ✨ امکانات جدید این نسخه
- 🎤 **چت صوتی ساده** — دکمه‌ی میکروفون بالای هر روم؛ فقط کسایی که توی همون روم هستن صداشون رو می‌شنون
- 🎟️ **پک استیکر** — دکمه‌ی استیکر کنار دکمه‌ی عکس، چند پک ایموجی بزرگ آماده
- 📱 **رفع باگ موبایل** — منوی کناری حالا روی موبایل با دکمه‌ی ☰ باز می‌شه (قبلاً قفل می‌شد)
- 🌐 **نمایش کل آنلاین‌های سایت** — بالای صفحه و توی پنل اعضا
- 🗑️ **پیام‌ها بعد از ۲۴ ساعت خودکار حذف می‌شن** — برای پاک نگه‌داشتن دیتابیس
- 🔑 **پنل مدیریت فقط با کد مخصوص** — دیگه برای همه باز نیست
- 🔒 **روم‌های خصوصی حالا دقیقاً مثل تلگرام کار می‌کنن** — توی لیست عمومی دیده نمی‌شن؛ فقط با وارد کردن کد دعوت (دکمه‌ی «🔑 ورود با کد» کنار روم‌های خصوصی) می‌شه واردشون شد
- ⚙️ **مدیریت روم برای سازنده‌ش** — هرکی یه روم بسازه، یه دکمه‌ی چرخ‌دنده (⚙️) بالای اون روم می‌بینه که می‌تونه ازش: اسم/آیکون روم رو عوض کنه، کد دعوت روم خصوصی رو عوض کنه (کد قبلی بی‌اثر می‌شه)، یا کل روم رو حذف کنه. ادمین‌های سایت هم به همه‌ی روم‌ها این دسترسی رو دارن.
